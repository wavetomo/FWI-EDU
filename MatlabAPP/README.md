# Application Overview

## 1. Record2dView

**Record2dView** is a seismic data visualization tool designed for displaying and processing 2D seismic records. Its main features include:

1. Supports reading seismic data in **Seismic Unix (.su)** format and **MATLAB (.mat)** format.
2. Provides data normalization and amplitude clipping (clip) functions to enhance signal visualization.
3. Displays seismic trace header information.
4. Allows exporting data in **binary, SU, and SEG-Y** formats for further processing and sharing.

---

## 2. SurveyDesign2D

**SurveyDesign2D** is a program for designing 2D seismic acquisition systems.

### Main Features

1. Supports multiple types of 2D acquisition geometries, including:
   - Surface acquisition
   - VSP (Vertical Seismic Profiling)
   - Crosshole acquisition
   - Surrounding acquisition

2. Visualization capabilities:
   - Geological model display
   - Source locations
   - Receiver locations

3. Acquisition parameters can be exported in **JSON** text format for storage and later use.

# Installing the MATLAB Application

1. Open MATLAB.
2. Click **“Apps”** in the top navigation bar.
3. Select **“Install App”**.
4. Browse to the folder containing the `.mlappinstall` file and select it.
5. Click **“Install”** to add it to *My Apps*.

After installation, the application will appear in the MATLAB Apps list and be ready for use.
